SpecGate 静态 HTML 生成与修复闭环

SpecGate 读取任务规约和检查清单，用可注入的 MockLLM 产生严格 JSON 动作， 经过白名单工具与 Guardrail 执行，再用静态 HTML Gate 反馈修复结果。

知识图谱 Demo 查看 MockLLM 生成的静态 HTML 页面：搜索知识点、点击模块并查看右侧详情。 运行报告 查看一次 harness 运行的 Gate 结果、检查项、通过状态和最终产物链接。 GitHub 仓库 查看 SPEC、PLAN、AGENT_LOG、测试、CI、Dockerfile 和完整实现代码。
核心输入：TASK_SPEC.md + CHECKLIST.md + 当前 index.html
核心机制：Action Parser、Workspace Guardrail、Tool Dispatcher、HTML Gate、Feedback Loop、Trace / Report。
